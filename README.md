# Keyboard Wizard CE

Community Edition / unofficial port of Modern KeyWizard for Minecraft 1.16.5 on Forge.

- Branch: `ce/forge-1.16.5`
- Mod id: `keyboard_wizard_ce`
- CE version: `1.0.1`
- Current author: WuXiaoYa
- Original authors: MrNerdy42, Tapio
- Art assets: WI_8614_ice
- License: AGPL-3.0-only
- Upstream project: Keyboard Wizard
- Target: Minecraft `1.16.5` + Forge `36.2.39+`

## Current Branch Features

- F7 opens Keyboard Wizard CE directly in-game.
- Main keyboard, numpad, and auxiliary key layouts are available from the CE screen.
- Middle-clicking a key filters the left-side list to bindings assigned to that key.
- The left-side key binding list shows the owning mod/category for easier identification.
- The vanilla Controls screen has a bottom-right toggle button for Keyboard Wizard CE.
- The CE screen uses the packaged community background texture and icon.
